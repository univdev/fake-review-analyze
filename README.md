# 11번가 리뷰 크롤러

## 설명

사용자가 입력한 상품 URL과 옵션에 따라 11번가 상품 리뷰를 자동으로 크롤링하여 저장하는 도구입니다.

## 지원 웹사이트

- [11번가](https://www.11st.co.kr)

## 사용 기술

- [Playwright](https://playwright.dev/) (웹 자동화 및 크롤링)
- [Commander](https://github.com/tj/commander.js/) (CLI 옵션 파싱)
