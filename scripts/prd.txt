# Fake Review Analyzer - Product Requirements Document

## Project Overview
The Fake Review Analyzer is a Python-based tool designed to crawl and analyze product reviews from major Korean e-commerce platforms (Coupang and Naver Shopping). The tool aims to help identify patterns and characteristics of fake reviews by collecting and analyzing review data.

## Core Features

### 1. Multi-Platform Review Crawling
- Support for multiple e-commerce platforms:
  * Coupang
  * Naver Shopping
- Unified crawling interface for consistent data collection
- Flexible URL parsing and validation
- Configurable page navigation and review collection

### 2. Data Collection & Storage
- Comprehensive review data model including:
  * Review text content
  * Rating information
  * Image attachments
  * Reviewer metadata
- CSV export functionality for data analysis
- Error handling and retry mechanisms
- Progress tracking and reporting

### 3. Technical Requirements

#### Development Environment
- Python 3.8+
- Selenium WebDriver for dynamic content
- BeautifulSoup4 for HTML parsing
- Environment variable management with python-dotenv

#### Core Dependencies
- selenium==4.15.2
- selenium-wire==5.1.0
- beautifulsoup4==4.12.2
- webdriver-manager==4.0.1
- python-dotenv==1.0.0
